Pre-design · pre-design

/twt-pre-design

Run the full pre-design phase — content, brand, positioning, IA, curation — into one handoff brief.

Version 1.2.2 Arguments: yes

Purpose

Runs your entire pre-design phase — pulling in content, then defining brand, positioning, structure, and a content plan — and rolls it all into one brief ready for design. It's the fastest path from messy project notes to a solid foundation.

Skills it runs, in order

This skill is an orchestrator — it dispatches the skills below (via the Agent tool) rather than doing their work itself. Here's each one, in the order it runs, and why.

  1. /twt-content-fetchPulls in any existing copy first, so every later step works from real material instead of guesses.
  2. /twt-brandDefines the brand voice and visual cues the rest of the site leans on.
  3. /twt-specCaptures the north-star spec when the project needs an explicit one.
  4. /twt-positioningSharpens who the site is for and what to promote.
  5. /twt-ia-define or /twt-ia-validateLays out the sitemap and scope, then checks it for gaps.
  6. /twt-curation-define or /twt-curation-validateDecides what content to keep and maps it onto pages, then reviews the plan.

Parameters it can expect

What's provided (URLs, PDFs, docs, brand book, Figma); optional --from/--only flags

Result it provides

  • .twt-artifacts/pre-design/pre-design-brief.mdThe Phase-1 handoff brief — brand, positioning, structure, and content plan rolled into one design-ready document.

Boundaries

  • Doesn't do design, development, or QA (later phases)
  • Doesn't reproduce sub-area logic — dispatches each sub-area's orchestrator, or (for IA + curation, which have no standalone command) their *-define / *-validate sub-skills directly (rule 5)
  • The brief is a static synthesis, not a live transition skill

Success criteria

  • Each requested sub-area runs in order A → (B ∥ S) → D → E → C — Brand and Spec run concurrently (disjoint inputs and outputs); Positioning waits for both
  • --from <area> resumes from a sub-area; --only <area> scopes to one
  • pre-design-brief.md summarizes brand, the north-star spec/direction, positioning, IA, and curation with links to every detailed artifact
